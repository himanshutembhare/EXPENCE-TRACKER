# Personal Expense Tracker (Python)

A Python-based expense tracking system using SQLite database.

## Features
- Add and manage expenses
- Category-wise search
- Monthly expense report
- Export data to CSV
- Persistent storage using SQLite

## Technologies
- Python
- SQLite
- CSV

## How to run
python main.py
